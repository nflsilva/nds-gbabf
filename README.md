# nds-gbabf
=======================================

A Nintendo DS program capable of re-flashing some Gameboy Advance bootleg cartridges.

## Supported cartridges
* MSP55LV128
* MSP55LV128M


## What's next
* Make it faster
* Implement a simple GUI for game selection
* Add more supported cartridges
